# Deterministic synthetic goldens

Run `make goldens` to reproduce every `.mlsg` file byte-for-byte with the current
compiler/runtime. Normal goldens exercise planar, oblique, curved, two-sheet,
neighborhood-change, support-boundary, ordering/duplicates, and paired
translation cases. Files beginning with `degenerate_` are characterization-only
and intentionally contain nonzero status plus NaN normals; the acceptance
comparator rejects them.

These files were generated by the independent FP32 oracle. They are not observed
upstream captures.
